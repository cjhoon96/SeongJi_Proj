# 2. 추가된 기능들
## * 시작 메뉴
스타트 버튼 외 난이도 조절 버튼 추가 easy 변수, 다양한 버튼 관련 변수 추가
난이도 조절 버튼을 누를시 ez normal 로 변경 가능 easy에서는 3개 이상 한꺼번에 터트릴시 추가 시간 부여
만약 기존 120초 보다 이상의 추가 시간이 주어질 경우 시간은 추가된 그대로 흘러가되 미관상의 문제로 시간 칸을 오버하여 표기되지 않게 수정함
